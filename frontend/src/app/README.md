# App

La carpeta **App** contiene toda la lógica de la aplicación Angular.

Su organización sigue una arquitectura basada en responsabilidades, donde cada carpeta cumple un propósito específico.

## Estructura

```text
App
│
├── Core → Hace funcionar la aplicación.
│
├── Shared → Elementos reutilizables.
│
├── Layouts → La estructura de las páginas.
│
└── Features → Las funcionalidades de la aplicación.
```

> Cada una de estas carpetas posee su propio `README.md` con una explicación más detallada de su función y organización.
